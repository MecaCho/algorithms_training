# greedy alogrithms

贪心法， 称贪心算法、贪婪算法:在对问题求解时，总是做出在当前看来是最好的选择。

## 适用场景

简单地说，问题能够分解成 问题来解决， 问题的最优解能递推到最终问题的最优解。这种问题最优解成为最优结构。
贪心算法与动态规划的不同在于它对每个子问题的解决方案都做出选择， 不能回退。
动态规划则会保存以前的运算结果，并根据以前的结果对当前进选择，有回退功能。
