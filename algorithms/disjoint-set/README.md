# 基本操作

• makeSet(s):建一个新的并查集，其中包含 s 个单元素集合。
• unionSet(x, y):把元素 x 和元素 y 所在的集合合并，要求 x 和 y 所在的集合 相交，如果相交则 合并。
• find(x):找到元素 x 所在的集合的代表，该操作也可以 于判断 两个元素是否位于同 个集合，只要将它们各 的代表 较 下 就可以 。
